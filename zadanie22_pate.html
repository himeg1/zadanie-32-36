Patej: HTML to język znaczników służący do tworzenia stron internetowych. Używa się go do definiowania struktury dokumentu. Przeglądarki internetowe interpretują kod HTML i wyświetlają jego zawartość. HTML działa w parze z CSS i JavaScriptem, tworząc podstawę stron WWW.

Patej: CSS to kaskadowe arkusze stylów. Pozwalają one na oddzielenie treści od wyglądu. Dzięki CSS można zmieniać kolory, rozmiary i położenie elementów. CSS zwiększa możliwości projektowania interfejsów użytkownika.

Patej: JavaScript to język skryptowy działający po stronie klienta. Pozwala na interaktywność stron internetowych. Można za jego pomocą tworzyć animacje, walidacje formularzy i dynamiczne treści. JavaScript współpracuje z HTML i CSS w nowoczesnych aplikacjach webowych.

Patej: PHP to język skryptowy działający po stronie serwera. Pozwala na tworzenie dynamicznych stron WWW. W połączeniu z bazą danych, jak MySQL, pozwala tworzyć systemy logowania, sklepy internetowe i fora. PHP jest bardzo popularny wśród twórców backendu.
